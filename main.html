ProSpeak

ProSpeak is a desktop application that takes text prompt in any language and genrates E-mail's in the conditions/context it has been given in acordance to the prompt, Or voice prompt in 6 dialects across 7 langauges.

ProSpeak was developed to cater for the non native english speakers working in corprate envirnoments, This was developed to break the lanaguge barrier in the work setting and to present one self more commuicatable.

The app is run using node with electron to render the application, It uses micro-apps built with python for voice trasncription and translation along with the genration, The genration part uses the text-davinci-003 LLM by OpenAi.

Ai Career Counselor

Typical identification tests are mcq with very limited spectra of answers, which becomes very difficult to answer without compromising on some crucial details as well as these traditional test doesn't lead on a question the questions are pre-defined and static

This website sought to clear all the mentioned problems by allowing the users to convey their entire answer by eliminating rigid mcq and implementing a text field. Users can convey their entire point and answer conflicting questions, The website also asks questions dynamically meaning it comprehends your previous answers to ask new ones to make the result more accurate. Users also can inform the bot if they are not satisfied with the prediction leading to the bot asking more questions to refine its answer, users can ask about the predictions, bot also can explain the professions it has curated in detail.

The website is run on node js as its backend, the webpages are all in html and css along with Bootstrap for position and gsap for animations. The bot is run on the GPT-3.5_turbo-16k model from OpenAi in a chat configuration configured with system prompts.

Ai Workflow Editor

This is a webiste which allows users to develop Ai workflows using nodes.

The premise of this website is to allow users to make Ai workflows using nodes and host it as an api, and use it on api calls made by other itegrations like IFTTT etc, Allowing non programmers to develop workflows to automate daily and tedious task like, Email Writing, Meeting Minutes making, Class notes making, Homework doing etc etc.

As mentioned this is still under development but the final version is expected to vector database, langchain, LLAMA, GPT-3, GPT-4, Bard, IFTTT, Gmail, Github and a wide verity of integrations.

The current stance of the project is that the Node Editor, Serialiser, Code Genrator has been built.

The website uses node as backend, The node editor, Serialiser and the genarator is completly built in vannila js with Konva.js which is a graphics library. The webpge is written in html and css along with Bootstrap.
